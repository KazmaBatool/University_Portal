# FAST-EduPortal Student Dashboard

A responsive and professional **Student Dashboard / Learning Management System (LMS)** built with **PHP, HTML, CSS, and JavaScript**.

---

## Features

- ✅ **User Authentication** – Login system with session management and secure logout.
- ✅ **Sidebar Navigation** – Collapsible sidebar with links to Dashboard, Courses, Assignments, Grades, Resources, Messages, Calendar, and Settings.
- ✅ **Hamburger Menu** – Sidebar toggle for mobile devices.
- ✅ **Student Info Banner** – Displays student name, program, semester, GPA, and progress.
- ✅ **Active Courses** – Shows course codes, titles, instructors, credits, progress bars, and status.
- ✅ **Performance Stats** – Track assignments, quizzes, and project completion.
- ✅ **Upcoming Deadlines** – List of pending tasks and dates.
- ✅ **Recent Activities Timeline** – Displays recent submissions, grades, and instructor feedback.
- ✅ **Responsive Design** – Fully optimized for desktop and mobile with modern UI/UX.
- ✅ **Professional Styling** – Clean and attractive dashboard using CSS and subtle animations.
## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript  
- **Backend:** PHP  
- **Session Management:** PHP Sessions  
- **Responsive Layout:** Flexbox, CSS Grid, Media Queries
